# kassis_fileprocessor
kassis シリーズ用ファイル処理サービス

## 概要

Office系ファイル(xlsx,odp,docxなど)やPDFなどを処理し、データのインポートや全文検索(Elasticsearch)に投入するサービスです。
他の kassis サービスから RabbitMQ 経由で処理メッセージを受け取り、処理を実行します。
